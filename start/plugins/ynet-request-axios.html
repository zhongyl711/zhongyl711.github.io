# axios请求处理

ynet-request-axios使用axios库处理http请求，导出了一个RequestByAxios类，一个getJSON方法

# RequestByAxios类

方法 参数 返回值
send: 发送请求 config: RequestConfig<any, any> 格式详见下方RequestConfig
callBack: (state: RequestState, content?: any) => void 回调函数，state详见下方RequestState，content为请求成功后返回的数据		 取消请求函数

RequestConfig：

type RequestConfig<I, O> = {
    [key: string]: any;
    /**请求路径 */
    url: string;
    /**请求类型 */
    method?:
        | "get"
        | "GET"
        | "delete"
        | "DELETE"
        | "head"
        | "HEAD"
        | "options"
        | "OPTIONS"
        | "post"
        | "POST"
        | "put"
        | "PUT"
        | "patch"
        | "PATCH"
        | "purge"
        | "PURGE"
        | "link"
        | "LINK"
        | "unlink"
        | "UNLINK";
    /**请求配置参数 */
    data: I;
    /** 文件（文件上传相关） */
    file: any;
    /** 文件名称（文件上传相关） */
    fileName: string;
    /**是否记录缓存 */
    enableCache?: boolean;
    /**缓存标识 */
    cacheId?: string;
    /**缓存时间（单位：毫秒） */
    cacheTimer?: number;
    /**httpHeader Http头*/
    headers?: Record<string, any>;
    /**路由跳转时保持请求 */
    keepWhenRouting?: boolean;
    /** 成功请求 */
    success?: (rspData: O) => void;
    /** 请求失败 */
    error?: (error: RequesterError) => Boolean | any;
    /**是否显示loading */
    loading?: boolean;
};

RequestState：

type RequestState = "connect" | "canceled" | string;

# getJSON

方法 参数 返回值
getJSON url: string JSON文件路径
config: AxiosRequestConfig = { timeout: 5000 } 详见AxiosRequestConfig 配置，超时默认5000ms		 Promise
interface AxiosRequestConfig {
  url?: string;
  method?: Method;
  baseURL?: string;
  transformRequest?: AxiosTransformer | AxiosTransformer[];
  transformResponse?: AxiosTransformer | AxiosTransformer[];
  headers?: any;
  params?: any;
  paramsSerializer?: (params: any) => string;
  data?: any;
  timeout?: number;
  timeoutErrorMessage?: string;
  withCredentials?: boolean;
  adapter?: AxiosAdapter;
  auth?: AxiosBasicCredentials;
  responseType?: ResponseType;
  xsrfCookieName?: string;
  xsrfHeaderName?: string;
  onUploadProgress?: (progressEvent: any) => void;
  onDownloadProgress?: (progressEvent: any) => void;
  maxContentLength?: number;
  validateStatus?: ((status: number) => boolean) | null;
  maxBodyLength?: number;
  maxRedirects?: number;
  socketPath?: string | null;
  httpAgent?: any;
  httpsAgent?: any;
  proxy?: AxiosProxyConfig | false;
  cancelToken?: CancelToken;
  decompress?: boolean;
  transitional?: TransitionalOptions
}

# 如何使用

# 1.项目package.json文件中导入@ynet/request-axios库

"dependencies": {
	"@ynet/request-axios": "1.0.0"
}

# 2.在项目中应用

基础应用详见起步》核心》请求 中的引入方式，getJSON使用方法如下

import { getJson } from "@ynet/request-axios";
getJson('****.json').then((jsonData) => {
	// 处理数据
}).catch(err => {
    // 处理错误
});
