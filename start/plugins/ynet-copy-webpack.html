# Ynet-copy-webpack

ynet-copy-webpack功能是将已经存在的单个文件或整个目录复制到构建目录。

# 如何使用

首先需要在项目中添加当前插件的依赖，yarn add xxx

在打包配置文件中的extendPlugin添加该插件

例如：

webpack.config.js

module.exports = {
	extendPlugin:[
        "@ynet/ynet-copy-webpack",
  ]
}
