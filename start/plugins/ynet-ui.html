# UI核心库

ynet-ui库提供双向绑定v-model、utils、form组件通用基础功能

# 如何使用

# 表单控件

表单组件依附原生的 form 元素，并针对 ajax 应用进行增强。Form 组件在初始化时会扫描表单所有表单元素，并根据类型实例化。

表单控件控制整个 Form 的生命周期，包括控件实例化、表单验证、提交等操作。

表单内控件使用方法及扩展方法请参考formCtrls

表单获取数据时只会获取带有name并可见的非禁用控件（包括隐藏域控件）。

# onBeforeSubmit

  • 提交前执行自定义方法，支持异步操作，param1 为 form 表单实例，param2 为表单控件实例， 如果返回 false，则终止提交

  • 应用场景： 点击提交按钮后会首先执行这个方法

 <ynet-form ref="form" url="sss" :onBeforeSubmit="onBeforeSubmit"></form>
onBeforeSubmit(form, ctrls) {
    //form 表单实例对象
    //组件实例对象
    return true; //返回false， 终止提交
}

# onEvaluate

  • 自定义数据转换方法，通过此方法可以在提交之前动态更改提交数据。

  • 若没有返回值，则按照 formData 提交

  • formData 为引用数据类型，如果添加属性，不 return 值也会添加成功

  • 应用场景： 数据转换、合并后提交

<ynet-form ref="form" url="sss" :onEvaluate="onEvaluate"></form>
onEvaluate(formData, form) {
    //formData  表单提交数据
    //form 表单实例对象
    return formData;
}

# onBeforeRequest

  • 表单提交前自定义方法，支持异步操作，param1 为 form 表单实例，param2 为 form 表单提交数据， param3 为表单控件实例， 如果返回 false，则结束提交

  • 应用场景: 进行签名，蓝牙 key 验签等

 <ynet-form ref="form" url="sss" :onBeforeRequest="onBeforeRequest"></form>
onBeforeRequest(form, formData, ctrls) {
    //form 表单实例对象
    //formData form 表单提交数据
    //表单控件实例
    return false; //返回false,则结束提交
}

# onSubmit

  • 自定义提交方法， 使用该该方法后将全面接管表单提交，配合 onEvaluate 方法使用；

  • 应用场景： 主要用于当前页面的表单不需要提交，到下个页面的时候和下个页面的表单数据合并提交

 <ynet-form ref="form" url="sss" :onSubmit="onSubmit"></form>
onSubmit(formData, form, ctrls) {
    //formData 表单数据
    //form 表单实例对象
    //ctrls 表单控件实例

}

# onReqSubmitKey

用于设置 submitKey 的自定义函数，param1 为 form 表单实例对象，返回值为 submitKey

 <ynet-form ref="form" url="sss" :onReqSubmitKey="onReqSubmitKey"></form>
onReqSubmitKey(form) {
    //form 表单实例对象

    return "123456"; //返回值为submitKey
}

# onSubmitSuccess

自定义提交成功方法，form 提交服务器成功后执行此方法

注意： rspData 属性是服务端返回数据后的 rspData 属性值

 <ynet-form ref="form" url="sss" :onSubmitSuccess="onSubmitSuccess"></form>
 //自定义提交成功后执行此方法
onSubmitSuccess(rspData, formData) {
    //rspData 请求返回数据
    //formData 表单数据
}

# onSubmitError

  • 自定义提交失败方法，form 提交服务器失败后执行此方法
<ynet-form ref="form" url="sss" :onSubmitError="onSubmitError"></form>
 //自定义提交成功后执行此方法
onSubmitError(rspData, formData) {
    //rspData 请求返回数据
    //formData 表单数据
}

# onShowErrorMessage

  • 自定义错误提示方式
<ynet-form ref="form" url="sss" :onShowErrorMessage="onShowErrorMessage"></ynet-form>
onShowErrorMessage(message, ctrls) {
    //message错误信息
    console.info(message);
}

# onCloseErrorMessage（目前只在 PC 端使用）

  • 自定义关闭错误提示方法
<ynet-form ref="form" url="sss" :onCloseErrorMessage="onCloseErrorMessage"></ynet-form>
onCloseErrorMessage(ctrl) {
    console.log(ctrl);
}

# 操作接口说明

# get

  • 获取控件的值
<ynet-form ref="form" url="sss"></ynet-form>
//获取控件的值
let formData = this.$refs.form.get();

# validate

  • 执行表单校验，支持单个校验和整个表单校验
<ynet-form ref="form" url="sss"></ynet-form>
//校验整个表单
this.$refs.form.validate().then(() => {});

//校验单个控件
this.$refs.form.validate("控件name").then(() => {});

# submit

  • 表单提交
<ynet-form ref="form" url="sss"></ynet-form>
//主动触发表单提交
this.$refs.form.submit();

# closeErrorMessage

  • 关闭错误提示消息
<ynet-form ref="form" url="sss"></ynet-form>
//表单项name 不输入name，则关闭表单内所有
this.$refs.form.closeErrorMessage("控件name");

# showErrorMessage

  • 显示错误信息
<ynet-form ref="form" url="sss"></ynet-form>
this.$refs.form.closeErrorMessage("控件name", "错误消息");

# getCtrls

  • 获取表单内所有表单控件
<ynet-form ref="form" url="sss"></ynet-form>
this.$refs.form.getCtrls();

# reset

  • 重置表单项
<ynet-form ref="form" url="sss"></ynet-form>
this.$refs.form.reset();
