# 持久化缓存

用于持久化本地存储（local）。

持久化缓存基于IoC实现，平台提供了默认的持久化缓存实现方案 ynet-local-storage，采用**window.localStorage**对数据进行持久化存储，除非主动删除数据，否则数据是永远不会主动销毁。

# API

方法 方法说明 参数 参数说明
get 根据key值获取对应缓存数据 key: string,
remove?: boolean		 key: 唯一键值
remove：获取后是否删除，默认false，可选参数。
set 根据key值设置缓存数据 key: string,
value: any,
key: 唯一键值
value: 储存任意值
remove 根据key值删除对应缓存数据 key: string key: 唯一键值
clear 清空所有缓存数据 无

# 示例

2.使用

// 在视图中
this.$app.localStorage.set("uuid", {name: "小明", age: "18"});
this.$app.localStorage.get("uuid");
this.$app.localStorage.remove("uuid");
this.$app.localStorage.clear();
// 不在视图中
app.localStorage.set("uuid", {name: "小明", age: "18"});
app.localStorage.get("uuid");
app.localStorage.remove("uuid");
app.localStorage.clear();
