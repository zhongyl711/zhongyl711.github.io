# YNET 核心介绍

当你看到本文前，请确保已完成了一个项目的搭建工作

YNET 平台提供两部分类型的功能：

  1. ynet 核心插件库，在此我们提供了所有终端所必需的功能性插件，其中包括脚手架、core 核心、工具库、请求、UI 核心库等插件
  2. 由于各个项目的 UI 样式及交互方式不同，我们建议各个分区内自主创建 UI 库，我们目前提供了 UI 库核心功能，后续我们将推出公司通用的 UI 组件库

注意：核心内容中提及的app对象，为Application实例，详见模板项目main.js文件

//初始化应用程序
window.app = new Application({
    //对接html中的Dom容器
    el: "#app",
    //路由配置
    routerOption: routerOption,
    //布局方案配置
    layout: DefaultLayout,
    //项目入口hash值，用于判断点击返回按钮是否执行onExit函数
    indexPath: "/home/index"
});
