# 依赖注入

平台引入依赖注入概念，以此实现实现类的可插拔，开发人员面向接口开发，无需关注具体实现逻辑，实现逻辑由项目启动时选择性注入

# API

方法 参数 备注
bind tag：symbol｜string 返回to方法，以置顶实现类
get tag：symbol｜string 通过tag返回实现

# 使用

我们通过操作Icontainer注入容器来进行操作

//注入会话缓存实现类(降级方案)
IContainer.bind(STORAGESESSIONINJECTTAGNAME).to(SessionStorage.checkEnable() ? SessionStorage : SessionWindow);

//注入持久化缓存实现类
IContainer.bind(STORAGELOCALINJECTTAGNAME).to(LocalStorage);

//注入请求管理实现类
IContainer.bind(REQUESTERCONFIGINJECTTAGNAME).to(SelfRequestConfig);



//操作
public config: RequesterConfig = IContainer.get(REQUESTERCONFIGINJECTTAGNAME);

该功能适用于高级开发，以提高项目实现解耦
