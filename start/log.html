# 日志

文件地址：ynet平台项目 ynet-core/src/common/log.ts

在项目开发过程中，我们常通过打印日志，进行调试，查看错误信息，并记录重要异常，以及重要过程，比如打印核心代码的运行逻辑过程等。

为了方便开发人员查看打印日志，我们对 console 进一步封装，统一打印格式，使输出方式更加人性化，便于查看。

# API

方法 参数 参数说明
info tagName: string
content: string		 tagName: 标签名字
content: 日志内容
warn tagName: string,
content: string		 tagName: 标签名字
content: 日志内容
error tagName: string
content: string
err?: Error		 tagName: 标签名字
content: 日志内容
error: 错误对象，可选参数

# 示例

import { Log } from '@ynet/core';
Log.info("理财购买", "成功")
Log.warn("理财购买", "成功")
Log.error("理财购买", "成功")
