1.联盟 6.0APPUI 组件库

该组件库是基于 YNET 平台提供的功能，根据联盟 6.0APP 的业务功能开发的 UI 组件库，包含基础组件，弹窗组件，表单组件，其它 UI 组件等。

此项目是基于 YNET 平台，联盟 6.0APPUI 组件库开发的测试示例，基于不同参数的组件展示不同效果的 UI 示例

YNET 平台提供了平台框架核心，打包，性能分析，数据储存，网络请求，表单校验等功能

# 开始创建一个项目

# 准备工具

  1. 安装包管理工具 Yarn

    参考链接：https://yarn.bootcss.com/docs/install/#mac-stable

  2. 下载模板工程

    在H5工程目录下，找到WAP端H5项目模板并进入工程，点击下载压缩包到指定目录并解压缩。

  3. 下载模板工程依赖库

    返回H5工程目录下，找到YNET平台并进入工程，克隆工程到指定目录。

# 标准步骤

  1. 使用开发管理工具打开下载的模板工程，工程下全局搜索关键字TODO，搜索结果会出现两类：TODO:(步骤序号)和TODO:(其他)。

    • **TODO:(步骤序号)**为项目初始化配置选项，可以按照关键字标记的步骤序号依次进行修改，以完成项目代码初始化操作的目的。

      1. 首先根据步骤序号，找到模板工程目录package.json文件下,修改工程名：name。

      2. 修改依赖库路径workspaces字段，次路径为模板工程目录下的相对路径，根据自己放置 ynet 依赖库目录修改即可。"../ynet/*"值为将ynet和模板工程放在同目录下。

      3. 添加依赖库，字段dependencies，例如："@ynet/cli": "1.0.0"。会将依赖库的文件夹名称 ynet-ui 进行自动转化为 ynet/cli。

      4. 然后根据步骤序号4，找到模板工程目录 /src目录下index.html文件,修改项目title，在title标签中，替换为项目名称。

      5. 最后根据步骤序号5，找到模板工程目录 /src/common/config目录下router.js文件,配置路由文件，目前默认的为 "/home/index"，首次建议不删除，跳过此步，待项目起始页展示后再进行修改删除。

    • **TODO:(其他)**为可选修改步骤，旨在帮助开发快速预览所有关键修改点，开发人员按需修改即可，不做强制要求。

  2. 打开终端命令行工具，到依赖库工程ynet下，运行命令yarn，等待安装。

  3. 安装成功之后，到模板工程目录下，运行命令yarn，安装工程依赖包。

  4. 在模板工程目录下运行 yarn run dev，成功自动打开浏览器展示项目
