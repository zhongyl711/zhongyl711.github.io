# 视图

该组件对Vue进行封装，对外暴露API，并对其进行封装管理。

# API

方法 参数
mixins 参考Vue.mixin( mixin ) (opens new window)
filter 支持原始Vue.filter( id, [definition]) (opens new window),及单个参数对象形式{name：String, filter: Function}
use 参考Vue.use( plugin ) (opens new window)

使用方式：操作提供的三个方法需要使用app.baseVue

注意事项：如果需要使用Vue的其它方法或是全局配置，请使用app.baseVue.Vue对象
