# 事件

文件地址：ynet平台项目 ynet-core/src/common/event.ts

该功能面相插件开发人员，业务开发人员可以跳过本节介绍

事件类基于发布订阅模式提供了on、off、once、trigger方法，插件开发者可以实现此类提供切面功能

# API

方法 方法说明 参数 参数说明
on 绑定指定名称的回调函数 name: string
callBack: EventCallBackType		 name: 绑定的唯一名称
callBack: 执行的回调函数，可以通过返回false终止后续回调执行
off 解除绑定当前回调函数，或者解除当前名字所有的回调函数 name: string, callBack?: Function name: 绑定的唯一名称
callBack: 执行的回调函数
once 绑定一个只执行一次的回调函数 同on方法 同on方法
trigger 触发指定名称的回调函数 name:string
param?: any		 name: 绑定的唯一名称
param: 执行on绑定的回调函数传入的参数

# 示例:

此示例只是演示使用方式，具体功能实现方式可参阅 ynet-core/src/request/index.ts

// 第一步 引入核心库中EventHandler类
import { EventHandler } from "@ynet/core";
// 第二步 继承此类，使用方式按需而定
class Test extends EventHandler{
    name: string = ""
    constructor() {
        super();
    }
	public get name() {
        return this.name;
    }
	public set name(name) {
        if(this.trigger("check", name)) {
         	this.name = name
        }
    }
}

const t = new Test();
t.on("check", function(name) {
	return !!name;
})
